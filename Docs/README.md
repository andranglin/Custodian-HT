# Documentation Directory

Custodian-HT documentation, examples, and playbooks.

## Subdirectories

- **examples/** - Usage examples and sample scripts
- **images/** - Screenshots and diagrams
- **playbooks/** - Threat hunting playbooks and investigation guides

## Playbooks

Guided investigation workflows for common attack scenarios:
- Ransomware detection and response
- Lateral movement hunting
- Persistence mechanism discovery
- Data exfiltration detection
- Credential access investigation
- Living-off-the-land technique detection
- Webshell hunting
- Business Email Compromise (BEC) investigation

## Examples

Sample scripts and command-line usage examples for:
- Quick triage collection
- Remote Windows hunting
- Remote Linux hunting
- OSINT investigations
- Memory analysis workflows
- Custom reporting

## Contributing

We welcome contributions of new playbooks, examples, and documentation!
